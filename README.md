Repaire du Drac — Plateforme JDR

Ce dépôt contient le cahier des charges et un premier socle du site (frontend +
API). Il sert de base pour la mise en place de la plateforme de gestion des
tables de Jeu de Rôle.

Document principal :
- docs/cahier-des-charges.md

Lancer le projet en local

1) Backend (API)
- cd backend
- npm install
- npm run dev

L’API répond sur http://localhost:4000/api/tables

2) Frontend (site)
- cd frontend
- npm install
- npm run dev

Le site sera disponible sur l’URL affichée par Vite.
